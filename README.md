# AnonyMessenger (рабочее название)

Анонимный мессенджер с открытым исходным кодом.
Клиент на Flutter, сервер на Go.

**Текущее состояние (MVP):**
- Регистрация по публичному ключу (Ed25519-подобная заглушка)
- Поиск пользователя по никнейму и добавление в контакты
- Отправка текстовых сообщений в реальном времени
- Хранение сообщений на сервере (SQLite), доставка при онлайне и подгрузка истории
- Тёмный современный интерфейс

**Стек:**
- Клиент: Flutter (Dart)
- Сервер: Go + gorilla/websocket + modernc.org/sqlite
- Ключи: временно случайные байты (для совместимости с flutter_secure_storage)
- Шифрование: отсутствует (сообщения передаются открытым текстом)

**Запуск:**
1. Сервер: `cd server && go run ./cmd/server`
2. Клиент: `cd client && flutter run`

**Будущие шаги (Roadmap):**
1. Внедрение End-to-End шифрования (Signal Protocol через `libsignal` FFI или X3DH + Double Ratchet)
2. Поддержка анонимной маршрутизации через Tor/I2P
3. Групповые чаты
4. Голосовые сообщения, вложения
5. Самоуничтожающиеся сообщения
6. Федерация/децентрализация серверов
